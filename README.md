# GestorUsuarios_CRUD
📌 CRUD de Usuarios con PHP y Bulma Aplicación web desarrollada con PHP y Bulma que permite gestionar usuarios mediante operaciones CRUD (Crear, Leer, Actualizar y Eliminar). Ofrece una interfaz simple, responsiva y fácil de usar.

✅ Funcionalidades principales
- Inicio de sesión con validación segura
- Manejo de sesiones
- Envío de formularios con AJAX
- Alertas dinámicas con SweetAlert
- Redirecciones controladas

🚀 Tecnologías utilizadas:
- PHP (Programación backend)
- MySQL (Base de datos)
- JavaScript (Fetch API + AJAX)
- SweetAlert2 (Alertas dinámicas)
- HTML5 / CSS3

Configurar credenciales. Crea un archivo llamado: server.php

- const DB_SERVER = "localhost";
- const DB_NAME = "base_de_datos";
- const DB_USER = "usuario_base_de_datos";
- const DB_PASS = "clave_base_de_datos";
